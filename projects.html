Survival Game

This game was created during a summer camp project. Requirements included multiple difficulty stages, keyboard movement, and a minimum of sixty seconds of gameplay. The final version includes three difficulty stages and a total of eighty seconds of survival time.

This project marked the beginning of my journey learning the C++ programming language. It was exciting to build a complete functioning game from scratch with only a small number of lines of code.

Robot Project

I built this robot in my freshman year for a LEGO Mindstorms competition. My goal was to create a robot that could grab and push opponents out of a sumo-style ring using claw-like arms.

This project introduced me to the C programming language. I programmed one Mindstorm unit to remotely control the other, similar to a drone.

The Explorer Game

The Explorer was my final summer camp project. It includes three levels, each at least sixty seconds long, and ends with a two-minute boss fight. The levels include a Flappy-Bird-style map, a survival challenge, and a final boss encounter.

This game represents everything I learned about C++ during the camp and was my largest project at the time.
