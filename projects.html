Survival Game

  • Description:I created this survival game for a project assignment during a summer camp. The project requirements were that the game should be at least sixty seconds long, have multiple stages or difficulty levels, the user should be able to move the character by using the keys on the keyboard, and, most importantly, it must be a simple design and very challenging. So I built a game where the player can move the character with the arrow keys on the keyboard, must survive for eighty seconds, and the stage of difficulty changes after every twenty seconds have passed while they play the game.

  • This project represents the beginning of my journey in learning the programming language of C++. During the summer camp, working on this project brought me overwhelming excitement because of how simple it was to build a minigame with very few lines of code involved with the program, which surprised me.

    • Robot

    • Description: I made the robot during my freshman year in high school. The challenge was to build and program a robot that can move and challenge other competing robots while using the LEGO Mindstorms. I used two LEGO Mindstorms, one for competing and one to control the other LEGO Mindstorm. I built the one for the competition with claws, so it can grab and drag the opposing LEGO Mindstorm robot out of the arena ring. I programmed the second LEGO Mindstorm to send signals to my other LEGO Mindstorm, so I could control the arena built robot like a gladiator drone.

    • The” build a robot for sumo challenge” was my first introduction to the programming language of C, also known as the programming language of C++'s older brother. Eventhough the challenge gave the winner a prize of their choosing, I mostly participated to further develop my skill of programming with C, to build something for fun.

    The Explorer

    • I created this mini adventure game called The Explorer for another project assignment during summer camp. The project required a minimum of three levels, with each level being at least sixty seconds long, player should be able to control the character on screen, and a boss battle must be embedded in the game. I created three different levels, each representing a difficulty. The first level I created is a game that is similar to a game called Flappy Birds. The second level is an updated version of the survival game I created, and the third level is a boss level where the player has to survive for two minutes.

    • The Explorer game was my last project during the summer camp. The adventure mini game was developed using the programming language of C++ as well. This final project I made during the last day of the summer camp, also demonstrates all the knowledge I learned about C++ during my journey within a short period of time.
